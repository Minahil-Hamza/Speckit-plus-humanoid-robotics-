# Physical AI & Robotics - Docusaurus Documentation

## About

Docusaurus provides a structured, searchable way to explore all 80 chapters of the Physical AI & Robotics book across 10 specialized modules.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window at `http://localhost:3000`. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Serve Production Build

```bash
npm run serve
```

## Deployment

This Docusaurus site is deployed alongside the main application on Vercel.

Using SSH:

```bash
USE_SSH=true npm run deploy
```

Not using SSH:

```bash
GIT_USER=<Your GitHub username> npm run deploy
```
